### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（患者）、医生、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告信息
5. 网站介绍
6. 商业合作
7. 医生
8. 个人中心
9. 医生收藏
10. 医生预约挂号订单
11. 病例管理


**医生功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 公告信息
5. 患者管理
6. 病例管理
7. 药品管理
8. 医生留言管理
9. 医生预约挂号订单管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（公告类型管理、科室管理、时间段管理、药品类型管理、职位管理）
4. 公告信息管理
5. 患者管理、病例管理
6. 单页数据管理
7. 药品管理
8. 医生管理、医生留言管理、医生收藏管理、医生预约挂号订单管理
9. 轮播图信息
